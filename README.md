# clojure-objc-sample

Sample clojure-objc project.

## Usage

Edit project.clj with your j2objc and clojure-objc locations.
Edit the sample xcode project Build Settings as well (header search path and library search path).

     lein compile
     lein objcbuild

     Run xcode project!
